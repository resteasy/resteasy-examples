JS-API
=================
Example of using RestEasy with JS-API:

System Requirements:
-------------------------
* 
* Maven 2.0.9 or higher

Building the project:
-------------------------

```bash
mvn clean install
```

Running the project and manually testing it:
-------------------------

```bash
mvn jetty:run
```

Open a browser at the following URL:

> [http://127.0.0.1:8080](http://127.0.0.1:8080)
